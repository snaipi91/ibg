МЕЖДУНАРОДНЫЙ АЭРОПОРТ «УФА»
ул.Аэропорт, село Булгаково, Уфимский район, Руспублика Башкортостан
Подрядчик:
ООО «Трансстроймеханизация» (ТСМ)
Срок исполнения:
2014-2015 гг.
В рамках проекта будет произведен восстановительный ремонт искусственной взлетно-посадочной полосы (ИВПП) № 2 с заменой плит покрытия; реконструкция существующего и строительство нового перронов для обеспечения эксплуатации расчетных типов воздушных судов (ВС). Проектом предусмотрено строительство новой соединительной рулежной дорожки, также будет произведена модернизация светосигнального оборудования РД, реконструкция водосточно-дренажной системы, строительство очистных сооружений, реконструкция системы электроснабжения аэродрома и внутриаэродромных автодорог, включая патрульную дорогу.
